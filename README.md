# 推荐系统科研训练笔记

本仓库用于记录推荐系统科研训练过程中的学习计划、每日进展、课程作业和知识整理。

## 目录结构

- `learning_plan.md`：学习计划
- `daily_logs/`：学习记录
- `assignment/`：作业记录
- `knowledge_base/`：个人知识库

## 主要内容

### 学习计划

在 `learning_plan.md` 中记录假期学习阶段安排。

### 学习记录

在 `daily_logs/` 中记录当天完成的学习内容、遇到的问题和下一步计划。

### 作业记录

在 `assignment/` 中保存课程作业记录

### 个人知识库

在 `knowledge_base/` 中分类整理长期使用的知识
