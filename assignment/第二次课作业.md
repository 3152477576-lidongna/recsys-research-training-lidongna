## 三、实验内容

本次实验使用 PyTorch 完成二维二分类任务。实验首先固定随机种子并生成240条二维样本，再将数据划分为训练集、验证集和测试集。随后使用 `TensorDataset` 和 `DataLoader` 按 Batch 读取数据，并使用单层线性模型完成二分类训练。

模型及训练设置如下：

| 项目 | 设置 |
|---|---|
| 模型 | `nn.Linear(2, 1)` |
| 损失函数 | `BCEWithLogitsLoss` |
| 优化器 | SGD |
| 学习率 | 0.2 |
| Batch Size | 16 |
| Epoch | 30 |
| 训练集、验证集、测试集 | 160、40、40 |

## 五、关键数据 Shape

| 变量               | Shape      | 含义               |
| ---------------- | ---------- | ---------------- |
| `features`       | `(240, 2)` | 240条样本，每条样本有2个特征 |
| `labels`         | `(240,)`   | 240条样本对应的类别标签    |
| `batch_features` | `(16, 2)`  | 一个Batch中的16条二维样本 |
| `batch_labels`   | `(16,)`    | 一个Batch中的16个标签   |
| `logits`         | `(16,)`    | 模型对16条样本输出的原始分数  |
| `loss`           | `()`       | 当前Batch对应的标量损失   |

## 六、关键概念

- **Tensor**：PyTorch存储和计算数据的基本形式，可以理解为能够参与模型运算的多维数组。
- **Batch**：一次送入模型进行计算和参数更新的一组样本。
- **Epoch**：模型完整学习一遍全部训练数据。
- **Logit**：模型经过Sigmoid函数之前输出的原始分数。
- **Loss**：衡量模型预测结果与真实标签之间差异的数值。
