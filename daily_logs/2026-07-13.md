
# 第二次课学习记录

## Part 1：Python 基础与 Tensor

### 学习内容

主要学习了 Python 基础语法和 PyTorch 中 Tensor 的基本操作。

Python 基础部分，主要学习了变量与基本数据类型，掌握了列表、元组、集合、字典和字符串等常用容器的特点及使用方法。 同时，我还学习了条件判断、循环、函数、类与对象等内容，理解了如何使用函数封装重复操作，以及如何使用类组织属性和方法。

Tensor 部分，主要学习了 Tensor 是 PyTorch 中存储数据、模型参数和计算结果的基本对象，掌握了 `shape`、`ndim`、`dtype`、`device` 和 `requires_grad` 等重要属性。还练习了 Tensor 的创建、索引、切片、形状变换、转置、增加或删除维度、逐元素运算、矩阵乘法、广播、聚合、拼接和数据类型转换等操作。

### 遇到的问题

学习过程中，我对列表、元组、集合和字典的区别不够清晰，容易混淆它们的访问方式和是否可以修改。

在 Tensor 操作中，我对 `shape` 和 `ndim` 的含义、逐元素乘法与矩阵乘法的区别，以及 `reshape`、`transpose`、`unsqueeze` 和 `squeeze` 的作用理解得不够准确。 

### 怎么解决的

我通过对比不同容器的有序性、可修改性、是否允许重复和访问方式，逐步明确了它们各自适合保存什么类型的数据。

对于 Tensor，我结合具体示例观察每次操作前后的 Shape。例如，使用 `reshape` 重新组织数据形状，使用 `transpose` 交换维度，使用 `unsqueeze` 增加 Batch 维度，使用 `squeeze` 删除长度为 1 的维度。同时，我通过打印 `shape`、`ndim` 和 `dtype`，核对每一步的数据结构。

## Part 2：机器学习与深度学习基础概念

### 学习内容

本部分主要学习了机器学习和深度学习中的任务类型、训练流程、模型类型及评价方法。

我了解了回归、分类、排序、序列预测和生成等常见任。我还学习了训练闭环中的样本、特征、标签、参数、前向传播、Logit、Loss、梯度、优化器和评价指标等核心概念 。此外，我学习了训练集、验证集和测试集的不同作用，理解了数据泄漏、欠拟合、过拟合和泛化能力等概念。我还初步了解了线性模型、决策树、MLP、CNN、RNN、LSTM 和 Transformer 的基本特点及其在推荐系统中的应用。

### 遇到的问题

我最开始容易混淆参数和超参数，也不清楚训练集、验证集和测试集为什么需要分别设置。

对于 Loss 和评价指标，我一开始认为它们都是用于衡量模型效果，没有理解二者在训练过程中的不同作用。同时，我对线性模型、MLP、CNN、RNN 和 Transformer 的适用场景缺少整体认识。

### 怎么解决的

我通过训练流程进行区分：模型参数是在训练中根据梯度自动更新的，例如权重和偏置；超参数是在训练前设置的，例如学习率、Batch Size 和 Epoch。

我理解到训练集用于更新模型参数，验证集用于选择模型和调整方案，测试集只用于最终评价。测试集不能参与训练和调参，否则会产生数据泄漏。

我也进一步明确了不同模型的特点：线性模型适合学习线性关系，MLP能够学习非线性特征组合，CNN关注局部模式，RNN 和 LSTM适合处理顺序信息，Transformer通过 Attention 建模长距离关系。
