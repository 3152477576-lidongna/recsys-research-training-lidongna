# 个人学习目标

## 一、总体目标

通过本次暑期科研训练，系统补充推荐系统、机器学习和深度学习基础，掌握论文阅读、代码理解、实验复现和结果分析的基本方法。

## 二、基础能力目标

### 1. 编程与工具基础

- 巩固 Python 基础语法和常用数据处理方法。
- 掌握 PyTorch 模型训练的基本流程。
- 熟悉 Git、GitHub、Obsidian、conda、SSH 等科研工具。
- 能够独立完成实验环境配置、依赖安装和基本报错排查。

### 2. 推荐系统基础

- 理解用户、物品和交互数据的基本含义。
- 掌握召回、排序、Top-K 推荐、隐式反馈等基础概念。
- 了解常见推荐模型的训练流程。
- 掌握 Recall、NDCG、HR、AUC 等常用评价指标的含义。

## 三、论文阅读目标

- 学会通过关键词、综述和代表性论文开展文献检索。
- 能够说明论文研究的问题、动机和主要创新点。
- 能够梳理论文的核心方法、主要模块和实验设计。
- 能够分析数据集、评价指标、基线方法和实验结果。

## 四、科研记录

- 坚持记录每日完成内容、遇到的问题和后续计划。
- 保存实验命令、参数配置、运行日志、报错信息和结果截图。
- 将重复出现的概念、报错和解决方法整理到个人知识库。
- 形成较清晰的论文阅读记录、代码复现记录和问题排查记录。

## 五、汇报表达目标

- 能够清楚介绍所研究的问题和论文背景。
- 能够概括论文的核心方法和主要创新。
- 能够说明自己运行了什么代码、使用了什么数据和参数。
- 能够展示实验结果并分析与论文结果的差异。
- 能够总结当前问题、已有进展和下一步计划。
